FAQ

Perguntas frequentes.

Preciso de uma conta Business no Instagram?
Sim. A Meta API só permite integração com contas Business ou Creator. Se você ainda tem uma conta pessoal, é só converter nas configurações do Instagram — leva 1 minuto e é grátis.
Como funciona o token Meta? É seguro?
O token é gerado no Graph API Explorer da Meta (developers.facebook.com) e dá permissão para nossa plataforma agir em nome da sua Página. Quando você cola o token, ele é imediatamente criptografado com AES-256-GCM antes de ir pro banco — nem nossa equipe consegue ler em texto plano. O frontend nunca vê o token novamente.
Posso cancelar quando quiser?
Sim. Sem fidelidade, sem multa. Cancele em 1 clique e seus dados ficam disponíveis para download por 30 dias.
Vocês postam Stories e Reels também?
Sim. Imagem, carrossel, Reels e Stories — tudo via Meta Graph API oficial. Para Reels e Stories, você precisa fornecer URL pública da mídia (suba no seu CDN, ImgBB, S3 ou similar).
Posso responder DMs pela plataforma?
Sim, desde que seu token tenha as permissões instagram_manage_messages e pages_messaging. A plataforma busca conversas a cada 30 segundos e suas respostas são enviadas em tempo real para o Instagram.
Vocês usam meus dados para treinar IA?
Não. Seus posts, métricas e mensagens são exclusivamente seus. Quando você usa a IA para gerar legendas, enviamos apenas o prompt — nenhum dado de outras seções trafega.
